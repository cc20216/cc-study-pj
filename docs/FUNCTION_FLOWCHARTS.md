# 智能学习管理系统 - 功能流程图

## 1. 用户登录流程图

```mermaid
flowchart TD
    A[用户访问系统] --> B{是否已登录?}
    B -->|是| C[跳转到首页]
    B -->|否| D[显示登录页面]
    D --> E[用户输入账号密码]
    E --> F[点击登录按钮]
    F --> G[后端验证用户信息]
    G --> H{验证成功?}
    H -->|是| I[创建用户会话]
    H -->|否| J[显示错误信息]
    J --> D
    I --> K[跳转到首页]
    K --> L[显示用户信息]
```

## 2. 作业提交流程图

```mermaid
flowchart TD
    A[用户访问进度页面] --> B[选择学习模板]
    B --> C[填写每日进度]
    C --> D[输入学习时长]
    D --> E[添加学习备注]
    E --> F[点击提交按钮]
    F --> G[后端验证数据]
    G --> H{数据验证成功?}
    H -->|是| I[保存进度数据]
    H -->|否| J[显示错误信息]
    J --> C
    I --> K[更新用户学习状态]
    K --> L[更新累计学习时长]
    L --> M[清除相关缓存]
    M --> N[跳转到进度看板]
    N --> O[显示提交成功信息]
```
